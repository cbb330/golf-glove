The Problem

When training athletes, golfing instructors must rely on their experience and eyes to notice errors in a golf student’s form and technique. A typical training session consists of a coach visually observing a player and then giving pointers and feedback based on their observations.

Our Solution

Golf is a glove and wrist assembly that tracks a player’s wrist position, as well as forearm position and grip strength. Users and coaches can record “swings” on a laptop-connected application and analyze the data to more fully understand an athlete’s needs. Position and movement of the wrist are tracked by using multi-resistive fabric that changes its resistance when stretched, while arm position is tracked using a wrist-mounted system that utilizes gyroscopes and accelerometers. Grip force is measured using a flexible force sensor.

Product Specifications

Features

  • Controlled by app
  • Sweat resistant and shock-proof
  • Replacement gloves for continued use
  • Adjustable wrist strap
  • Battery monitoring
  • Records swing to improve form

Technical Specifications

  • 20 feet Bluetooth communication between control device and unit
  • Minimum 2 hour battery life
  • Able to measure grip pressure
  • Gyroscope and accelerometer to record and replay forearm position
  • Detects wrist rotation and inflection up to 45 degrees

Applications

  • Gather real-time information about golf technique
  • System can be paired with autonomous application to give helpful tips to user
  • Record and replay previous golf swings for analysis

Source Code

Golf Glove source code is protected under a non-disclosure agreement between Golf Glove design team members and the project's supporting party. Because of this, source code for the project cannot be provided.

Datasheets

ComponentDescriptionDatasheet
Cypress CYW20719 Microprocessor present in the wrist unit that contains an integrated Bluetooth Low Energy (BLE) radio and an ARM Cortex M0 processor.
Cypress CYW920719Q40EVB-01 Integrated development board driven by Cypress CYW20719 microprocessor. Used for development process and debugging of design.
Texas Instruments ADS1115 Analog-to-Digital Converter (ADC) integrated circuit used to read variable resistance values from LiquidWire sensors.
Interlink 400 FSR Force Sensitive Resistor (FSR) sensors used to measure pressure applied to the golf club.
ST Microelectronics LSM9DS1 9-Axis Inertial Measurement Unit (IMU) used in both the glove and the wrist-mounted controller unit. Measures linear acceleration, angular rate, and magnetic field strength.
LiquidWire Sensor Linear soft robotic elastic sensors used to measure wrist flexion and extension.
